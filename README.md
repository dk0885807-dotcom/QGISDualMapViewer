# QGIS Dual Viewer

QGIS Dual Viewer is a QGIS plugin that provides two synchronized map
canvases for side-by-side visual comparison of spatial datasets.

## Features
- Two independent map canvases
- Real-time synchronized pan and zoom
- Separate layer panels for each map view
- Dockable and undockable map viewers
- Useful for change detection and visual comparison

## Requirements
- QGIS 3.34 or later

## Installation Steps
1. Download the QGIS Dual Viewer plugin ZIP file
2. Open QGIS
3. Go to **Plugins → Manage and Install Plugins**
4. Select **Install from ZIP**
5. Browse to the downloaded ZIP file
6. Click **Install Plugin**
7. Enable **QGIS Dual Viewer** from the Plugins menu

## Usage Steps
1. Open QGIS
2. Go to **Plugins → QGIS Dual Viewer**
3. Two map views will open side by side
4. In the **Layers Panel**, select the layer(s) you want to view in the new viewer
5. Click the **Right Arrow (→) / Add to Right Map** button
6. The selected layers will be added to the right map viewer
7. Add different layers to the left or right map as required
8. Pan or zoom in one map view
9. The second map view will automatically synchronize
10. Use dock/undock options to adjust the layout as needed

## Typical Use Cases
- Visual comparison of multi-date satellite images
- Change detection analysis
- Comparing classified and original datasets
- Side-by-side map interpretation
